📖 各占術のルーツ

読んで「へえ」となる豆知識

四柱推命中国 / 7〜11世紀

唐代に李虚中が基礎を作り、宋代の徐子平が「時柱」を加えて完成。年・月・日・時の四柱と天干地支の組み合わせは約518,400通りにも及ぶ。

マヤ暦占いメソアメリカ / 紀元前3000年〜

マヤの「ツォルキン暦」は260日サイクルの農業・儀礼暦。古代マヤは金星の会合周期を584日と計算し、現代値（583.92日）にきわめて近い精度で把握していた。

九星気学中国→日本 / 明治〜大正

中国古典「洛書（らくしょ）」の魔方陣が起源。1〜9の数字をどの方向に足しても15になる謎の配列が、星と方位の体系へと発展した。

タロット占いイタリア / 15世紀

もともとは貴族が楽しんだカードゲームだった。占いに使われ始めたのは18世紀フランスから。78枚の現在の形が確立したのは意外と最近、19世紀末のイギリス。

ルーン占い北欧・ゲルマン / 2000年以上前

「rūn」は古代語で「秘密・秘儀」を意味する。北欧神話では、オーディン神が9日間木に逆さ吊りになって自らを犠牲にし、ルーンを手に入れたとされる。

宿曜占星術インド→日本 / 9世紀

密教の流入とともに日本へ渡り、空海らが広めたとされる東洋占星術。平安貴族は「危宿（きしゅく）」には重要な決断を避け、出会いの宿を調べてから縁談を進めた。

ヒューマンデザイン現代 / 1987年

カナダ人Ra Uru Huがイビサ島で8日間の神秘体験を経て「受け取った」とされるシステム。易経・西洋占星術・カバラ・チャクラを一つに統合した最新の占術。

エニアグラム中東 / 起源不詳

グルジェフが20世紀初頭に中東から西洋へ伝えたとされる。「エニア」はギリシャ語で「9」。9タイプは人間の9つの本質的な欲求パターンを表すという。

MBTI性格診断アメリカ / 1940年代

ユングの心理タイプ論をマイヤーズ母娘が実用化。もともとは「戦時中の女性が自分に合う仕事を見つけるため」に開発された。現在は世界で年間200万人が受検。

バイオリズムドイツ・オーストリア / 19世紀末

医師フリース（1890年代）と心理学者スウォボダ（1900年頃）がほぼ同時期に独立して発見。身体23日・感情28日・知性33日という周期は、生まれた日をゼロ点に一生続くとされる。

動物占い日本 / 1998年

四柱推命の難解な理論をキャラクターで親しみやすく再設計した日本生まれの占術。12動物×60性格の組み合わせで約720通りの個性を表現する。

かずたま数秘占い日本 / 現代

「言霊（ことだま）」信仰から生まれた日本独自の数秘術。五十音の各音に1〜10の数霊が宿るとされ、名前の読みからエネルギーを読み解く。当サイトでは入門向け簡易版を使用。
